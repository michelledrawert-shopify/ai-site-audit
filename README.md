# AI Site-Audit Playbook

A repeatable playbook for Shopify Partner agencies: turn a prospect's name and
domain into an evidence-based commerce audit, a few sharp "why Shopify"
positioning hooks, and a ready-to-send outreach sequence that earns the first
conversation — using off-the-shelf AI and web tools.

Static site: plain HTML/CSS/JS, no build step.

## Contents
- `index.html` — the walkthrough and the reusable audit prompt
- `leave-behind.html` — printable one-doc leave-behind (toolchain cheat sheet, prompt, guardrails, templates)
- `ai-site-audit-playbook.pdf` — PDF export of the leave-behind
- `css/`, `js/`, `favicon.svg` — assets

## Deploy (GitHub Pages)
Settings → Pages → Source: Deploy from a branch → `main`, folder `/ (root)`.
Publishes at `https://michelledrawert-shopify.github.io/ai-site-audit/`.
Works the same on Vercel, Netlify, or Cloudflare Pages (root, no build command).

Asset paths are relative; `.nojekyll` disables Jekyll processing.
